# Shortest-python
Python library for (shorte.st)[http://shorte.st/].

## INSTALLATION
```
pip3 install git+https://github.com/CubexX/shortest-python.git
```

## USAGE
```python
from shortest import Shortest

url = Shortest.get('google.com', 'YOUR_TOKEN')
```
